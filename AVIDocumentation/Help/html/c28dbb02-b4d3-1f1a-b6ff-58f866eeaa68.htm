EFDataAccessEntityType Métodos

El tipo EFDataAccessEntityType expone a los siguientes miembros.

Métodos
 NombreDescripción
Create Añade un registro a la base de datos.
CreateFromList Añade todos los registros en una lista a la base de datos.
Delete Elimina un registro de la base de datos.
Refresh Lee todos los registros de la base de datos.
SaveAndRefresh Guarda todos los cambios y refresca el origen de datos. Además, devuelve la cantidad de ítems afectados.
