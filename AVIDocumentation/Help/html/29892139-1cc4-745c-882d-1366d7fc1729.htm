dataBaseView

DataGrid utilizado para mostrar los datos de la tabla.

Sus columnas se generan automáticamente a partir de las propiedades de EntityT a las que se les haya asignado un TextBox o ComboBox.

Las columnas se nombrarán de formas diferentes según si las columnas fueron generadas con un TextBox o con un ComboBox:

TextBox"El nombre de la columna es [nombre_columna].
ComboBoxHay dos columnas una de valor y otra de muestra. La de valor tiene el nombre [nombre_columna_id] y la de muestra el nombre "[nombre_columna_id]Display"
