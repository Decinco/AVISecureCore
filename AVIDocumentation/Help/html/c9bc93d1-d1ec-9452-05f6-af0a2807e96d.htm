AVISC_EFMaintenanceBaseFormEntityT, ContextT Clase

Formulario base para la creación de formularios de mantenimiento utilizando EntityFramework.

Se pueden añadir automáticamente campos a la tabla creando un TextBox o ComboBox en el formulario y luego introduciendo el nombre del tag con el formato correcto.

TextBox"[nombre_columna]"
ComboBox"[nombre_tabla_foránea].[nombre_columna_foránea].[nombre_columna_id]"
