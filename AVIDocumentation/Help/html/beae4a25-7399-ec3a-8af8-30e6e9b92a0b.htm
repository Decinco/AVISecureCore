DataAccess Métodos

El tipo DataAccess expone a los siguientes miembros.

Métodos
 NombreDescripción
Actualitzar Actualiza los datos en la base de datos a partir de un DataSet.
EncryptConfigFile Protege la sección de cadenas de conexión del archivo de configuración.
Executar Ejecuta una consulta SQL no relacionada con resultados (como INSERT, UPDATE o DELETE).
PortarDataset Obtiene todos los registros de una tabla específica.
PortarDatasetIForanies Obtiene todos los registros de una tabla y todas las tablas que tenga vinculadas a través de una clave foránea.
PortarInfoFK Obtiene la información sobre claves foráneas de la tabla indicada.
PortarPerConsulta(String) Ejecuta una consulta SQL y devuelve los resultados.
PortarPerConsulta(String, String) Ejecuta una consulta SQL y asigna un nombre a la tabla resultante.
